SEO

¿Qué es?

SEO son las siglas en inglés Search Engine Optimization que se traduce como Optimización para motores de búsqueda, y es una técnica implementada en páginas para incrementar su visibilidad y el posicionamiento en los resultados de los motores de búsqueda

Importancia

Es importante por la gran cantidad de tráfico en la red, y si se quiere llegar a ocupar los primeros lugares, usar las técnicas de SEO es una buena forma, así se va a poder destacar entre la competencia y atraer más visitantes a la página web, lo que aumenta la credibilidad

¿Cómo optimizar una página web para SEO?

Para optimizar una página web para SEO, se deben seguir varias estrategias:

  • Utilización de palabras clave
  • Tener una buena URL
  • Crear contenido de calidad y original.
  • Crea títulos únicos para cada página
  • Mejorar la velocidad de carga de la página
  • Usa una estructura adecuada usando correctamente las metaetiquetas
  • Utiliza imágenes optimizadas
Además, es importante realizar un seguimiento y análisis del rendimiento de la página para hacer ajustes y mejoras continuas

¿Qué son las Meta Etiquetas y describir las principales?

Las meta etiquetas se utilizan para describir el contenido de una página web, y se encuentran en el código HTML. Estas se encuentran siempre dentro del head. Los metadatos no se muestran en la página, pero los motores de búsqueda podrán analizarlos. Las principales son:

  • title: Es el título de la página que aparece en los resultados de búsqueda
  • description: Es un resumen breve del contenido de la página
  • keywords: Son palabras o frases clave relacionadas con el contenido de la página
  • charset: Define la codificación que utiliza la página
  • author: Muestra el autor de la página
  • viewport:Es importante para asegurar que las páginas se vean correctamente en dispositivos móviles

¿Cómo se aplica el tema en la página de esta investigación?

Este sitio implementa estructuras limpias, metaetiquetas y contenido relevante para facilitar su posicionamiento en motores de búsqueda.

Usabilidad Web

¿Qué es?

La usabilidad web es la facilidad que tiene el usuario para poder interactuar con ella, es decir, que sea intuitiva, que pueda encontrar fácil lo que quiere y que sea agradable

La usabilidad web mide 3 factores muy importantes

  • Eficicencia: El usuario logra el objetivo de la búsqueda
  • Eficacia: Buena usabilidad, logra el propósito rápido
  • Satisfacción: El usuario considera que la navegación fue fácil y agradable

Importancia

Una buena usabilidad es útil para mejorar la exeriencia del usuario y así aumentar la probabilidad de que la persona regrese o recomiende el sitio, y así la posibilidad de aumntar el número de visitas Una buena usabilidad web es útil para mantener a los usuarios interesados. Un sitio web fácil de usar puede aumentar la satisfacción del usuario, reducir la tasa de rebote y mejorar las conversiones Los principios de usabilidad web de Jakob Nielsen son la base de cualquier página web para que sea “user friendly” para que así los usuarios quieran ver la página, porque si el sitio es difícil de entender o lento, el usuario no va a querer quedarse.

Principios de Jakob Nielsen

  1. Visibilidad del estado del sistema: El usuario siempre debe de saber que es lo que esta pasando, por lo que página siempre le debe de estar dando alertas de lo que esta pasando

    Por ejemplo: Barra de progreso que va avanzando mientras carga el contenido

  2. Relación entre el sistema y el mundo real: Se tiene que hablar el mismo lenguaje que el usuario, se pueden utilizar imágenes o frases que sean fáciles y claras, usando un orden lógico

    Por ejemplo: El ícono de papelera para eliminar archivos

  3. Control y libertad del usuario: El usuario puede equivocarse, por eso siempre se le debe dar la posibilidad de revertir algo que haya realizado para que no se frustre

    Por ejemplo: Eliminar articulos del carrito

  4. Consistencia y estándares: Algunos íconos que para las personas ya es algo normal, los entienden y saben utilizar, por lo que si se cambian, puede empeorar la usabilidad ya que el usuario puede que no entienda con otros íconos

    Por ejemplo: 3 líneas horizontales que representan un menú desplegable

  5. Prevención de errores: Hay que prevenir cualquier error que el usuario podría cometer, hay que indicarle para que vea su error antes de que lo haga y si igual cae hay que buscar la forma de reconducirle de manera fácil y rápida

    Por ejemplo: Función de autocompletar de Google

  6. Reconocer antes que recordar: Es mejor que el usuario reconozca a que tenga que memorizar acciones para realizar algo

    Por ejemplo: Previsualizar las tipografías

  7. Flexibilidad y eficiencia de uso: EL sitio web debe de estar preparado para cualquier tipo de usuario, ya sea experto o novato

    Por ejemplo: Buscadores de google, si tienes experiencia se pueden usar operaciones para llegar al resultado más rápido, si no simplemente se busca lo que se quiere

  8. Diseño estético y minimalista: Debemos evitar elementos, información innecesaria al usuario, ya que esto distrae al usuario y hace que la pagina no cargue rápido

    Por ejemplo: El diseño del buscador de Google tiene solo lo imprescindible

  9. Ayudar a los usuarios a reconocer, diagnosticar y corregir los errores

    Por ejemplo: Cambiar el error 404 por un mensaje que se entienda más

  10. Ayuda y documentación: En algunas ocasiones los usuarios podrían llegar a ocupar un manual para lograr su objetivo

    Por ejemplo: La sección de preguntas frecuentes

Accesibilidad Web

¿Qué es?

La accesibilidad web es la práctica de diseñar y desarrollar páginas web que puedan ser usadas por todas las personas, incluidas aquellas con discapacidades como visuales, auditivas, motoras o cognitivas, agregando herramientas para que puedan navegar por todo el contenido.

Importancia

La accesibilidad web es importante para crear un internet sin barreras para que así las personas puedan acceder a la información que ocupan, sin importar sus habilidades o limitaciones, y esto mejora la experiencia del usuario

Principios fundamentales de la accesibilidad web según las WCAG

  • Perceptible: Todos los usuarios deben poder distinguir el contenido de la página, incluidos los que utilizan tecnologías de asistencia.

    Ejemplo: Texto alternativo para imágenes

    <img src="paisaje.jpg" alt="Paisaje montañoso con montañas y un sol">

  • Operable: Los componentes de la página como botones y formularios deben ser accesibles para el usuario.

    Ejemplo: Navegación por teclado

    Presiona Tab para navegar entre botones

  • Comprensible: La información y el funcionamiento deben ser fáciles, legibles y ayudar a no cometer errores.

    Ejemplo: Formulario con etiquetas claras


  • Robusto: El sitio web debe ser compatible con tecnologías actuales y futuras, navegadores y tecnologías de asistencia.

    Ejemplo: Uso de elementos semánticos
